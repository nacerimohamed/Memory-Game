```bash

git clone https://github.com/votre-username/memory-game.git

# 2. Accédez au dossier
cd memory-game

# 3. Ouvrez avec un serveur local (optionnel)
python -m http.server 8000

# 4. Ouvrez votre navigateur sur http://localhost:8000
