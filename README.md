## ✨ Fonctionnalités

### 🎯 Gameplay
- **8 paires de cartes** (16 cartes au total) avec des emojis d'animaux
- **Système de mémoire** : retournez deux cartes et trouvez la paire identique
- **Chronomètre** : le temps commence au premier mouvement
- **Compteur de mouvements** : suivez vos performances
- **Messages interactifs** : feedback en temps réel sur votre progression

### 🌍 Bilingue (FR/EN)
- Interface complètement traduite en **Français** et **Anglais**
- Changement de langue instantané sans perte de progression
- Messages de victoire adaptés à la langue choisie

### 🎨 Design
- Interface moderne avec dégradés et ombres
- Animation 3D lors du retournement des cartes
- Design responsive : fonctionne sur mobile, tablette et desktop
- Effets visuels pour les cartes appariées (surbrillance dorée)

### ⚙️ Techniques
- JavaScript Vanilla (sans frameworks externes)
- Gestion d'état complète (timer, moves, matches)
- Système de verrouillage pour éviter les clics multiples
- Algorithme de mélange Fisher-Yates
